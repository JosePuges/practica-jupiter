# 🪐 Práctica Júpiter

Este repositorio contiene un análisis exploratorio y técnico desarrollado en **Jupyter Notebook**. El proyecto se centra en el estudio de datos astronómicos, específicamente sobre el sistema de satélites de **Júpiter**, aplicando herramientas de Ciencia de Datos para la visualización y procesamiento de información.

---

## 🎯 Objetivos del Proyecto

* **Procesamiento de Datos:** Limpieza y transformación de datasets astronómicos.
* **Análisis Estadístico:** Cálculo de masas, radios y periodos orbitales de los satélites.
* **Visualización Científica:** Generación de gráficos para identificar patrones en las lunas galileanas.

---

## 🛠️ Stack Tecnológico

El análisis se apoya en las siguientes librerías de Python:

| Librería | Propósito |
| :--- | :--- |
| **Pandas** | Manipulación y limpieza de estructuras de datos. |
| **NumPy** | Operaciones matemáticas y cálculo numérico. |
| **Matplotlib** | Creación de visualizaciones estáticas. |
| **Seaborn** | Gráficos estadísticos de alta calidad. |
